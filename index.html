Introduction

This page is built as plain HTML/CSS with a LaTeX‑inspired look: paragraph indentation, numbered sections and subsections, equation numbers, and auto‑numbered figures/tables. It is lightweight and GitHub Pages–ready.

Inline math like \(x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}\) and display math below will be rendered by KaTeX automatically.

\[ e^{i\pi} + 1 = 0 \]
